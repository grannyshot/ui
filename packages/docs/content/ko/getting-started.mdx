# 시작하기

## 설치

```bash
npm install @grannyshot/ui
# 또는
pnpm add @grannyshot/ui
# 또는
yarn add @grannyshot/ui
```

## 셋업

### 1. 스타일 import

앱 루트에서 CSS 파일을 한 번만 import합니다:

```tsx
import '@grannyshot/ui/styles.css'
```

### 2. ThemeProvider 추가

다크 모드를 사용하려면 `ThemeProvider`로 감쌉니다:

```tsx
import { ThemeProvider } from '@grannyshot/ui'

export default function App() {
  return (
    <ThemeProvider defaultTheme="system">
      {/* 앱 내용 */}
    </ThemeProvider>
  )
}
```

### 3. Provider 추가 (선택)

명령형 API(toast, dialog, drawer)를 사용하려면 Provider를 추가합니다:

```tsx
import {
  ThemeProvider,
  ToastProvider,
  DialogProvider,
  DrawerProvider,
} from '@grannyshot/ui'

export default function App() {
  return (
    <ThemeProvider defaultTheme="system">
      {/* 앱 내용 */}
      <ToastProvider placement="bottom-end" />
      <DialogProvider />
      <DrawerProvider />
    </ThemeProvider>
  )
}
```

### 4. 컴포넌트 사용

```tsx
import { Button, Input, Field } from '@grannyshot/ui'

function LoginForm() {
  return (
    <form>
      <Field label="이메일" required>
        <Input placeholder="you@example.com" />
      </Field>
      <Button variant="primary">로그인</Button>
    </form>
  )
}
```

## Import 경로

| 경로 | 설명 |
|------|------|
| `@grannyshot/ui` | 전체 (컴포넌트, 스타일, 토큰, 유틸) |
| `@grannyshot/ui/react` | React 컴포넌트만 |
| `@grannyshot/ui/styles` | 스타일 recipe (cva) |
| `@grannyshot/ui/tokens` | 디자인 토큰 |
| `@grannyshot/ui/context` | ThemeProvider, useTheme |
| `@grannyshot/ui/utils` | cx 유틸리티 |
| `@grannyshot/ui/styles.css` | CSS 파일 |
