# grannyshot-ui

**Panda CSS**와 **Ark UI** 기반의 미니멀 디자인 시스템.

## 특징

- **26개 컴포넌트** — 버튼, 인풋, 모달, 토스트, 날짜 선택기 등
- **다크 모드** — 시맨틱 토큰으로 라이트/다크 자동 전환
- **접근성** — Ark UI(Zag.js 상태 머신) 기반 키보드 내비게이션 & 스크린 리더 지원
- **명령형 API** — `toast.success()`, `dialog.confirm()`, `drawer.open()`
- **트리 쉐이킹** — 필요한 것만 import, 소비자에게 Panda CSS 의존성 없음
- **TypeScript** — 완전한 타입 안전성

## 빠른 시작

```bash
npm install @grannyshot/ui
```

```tsx
import '@grannyshot/ui/styles.css'
import { Button, ThemeProvider } from '@grannyshot/ui'

export default function App() {
  return (
    <ThemeProvider>
      <Button variant="primary">클릭</Button>
    </ThemeProvider>
  )
}
```
