# Textarea

다양한 크기, 유효성 검증 상태, 리사이즈 옵션을 지원하는 멀티라인 텍스트 입력 필드입니다.

## Import

```tsx
import { Textarea } from '@grannyshot/ui'
```

## 사용법

```tsx
<Textarea placeholder="메시지를 입력하세요" />
```

## 크기

```tsx
<Textarea size="sm" placeholder="Small" />
<Textarea size="md" placeholder="Medium" />
<Textarea size="lg" placeholder="Large" />
```

## 상태

```tsx
<Textarea state="error" placeholder="에러 상태" />
<Textarea state="success" placeholder="성공 상태" />
```

## 리사이즈

```tsx
<Textarea resize="none" placeholder="리사이즈 불가" />
<Textarea resize="vertical" placeholder="세로 리사이즈 (기본)" />
<Textarea resize="both" placeholder="양방향 리사이즈" />
```

## 비활성화

```tsx
<Textarea disabled placeholder="비활성화된 텍스트영역" />
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `size` | `'sm' \| 'md' \| 'lg'` | `'md'` | 텍스트영역 크기 |
| `state` | `'default' \| 'error' \| 'success'` | `'default'` | 유효성 검증 상태 |
| `resize` | `'none' \| 'vertical' \| 'both'` | `'vertical'` | 리사이즈 방향 |
| `placeholder` | `string` | — | 플레이스홀더 텍스트 |
| `disabled` | `boolean` | `false` | 비활성화 상태 |
| `className` | `string` | — | 추가 CSS 클래스 |

네이티브 `<textarea>` HTML 속성을 모두 지원합니다.
