# Slider

Ark UI 기반의 범위 슬라이더 컨트롤로, 레이블과 값 표시 기능을 지원합니다.

## Import

```tsx
import { Slider } from '@grannyshot/ui'
```

## 사용법

```tsx
<Slider label="볼륨" defaultValue={[50]} />
```

## 크기

```tsx
<Slider size="sm" label="Small" defaultValue={[30]} />
<Slider size="md" label="Medium" defaultValue={[50]} />
<Slider size="lg" label="Large" defaultValue={[70]} />
```

## 값 표시

```tsx
<Slider label="밝기" showValue defaultValue={[75]} />
```

## 범위 설정

```tsx
<Slider label="가격" min={0} max={1000} defaultValue={[200]} />
```

## 비활성화

```tsx
<Slider label="비활성화" defaultValue={[50]} disabled />
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `label` | `string` | — | 슬라이더 레이블 텍스트 |
| `size` | `'sm' \| 'md' \| 'lg'` | `'md'` | 슬라이더 크기 |
| `showValue` | `boolean` | `false` | 현재 값 표시 여부 |
| `defaultValue` | `number[]` | — | 초기 값 |
| `min` | `number` | `0` | 최소 값 |
| `max` | `number` | `100` | 최대 값 |
| `disabled` | `boolean` | `false` | 비활성화 상태 |
| `className` | `string` | — | 추가 CSS 클래스 |
