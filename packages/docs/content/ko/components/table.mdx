# Table

데이터를 행과 열로 표시하는 테이블 컴포넌트입니다. Compound component 패턴으로 구성됩니다.

## Import

```tsx
import { Table } from '@grannyshot/ui'
```

## Usage

```tsx
<Table.Root>
  <Table.Header>
    <tr>
      <Table.HeaderCell>이름</Table.HeaderCell>
      <Table.HeaderCell>이메일</Table.HeaderCell>
      <Table.HeaderCell>역할</Table.HeaderCell>
    </tr>
  </Table.Header>
  <Table.Body>
    <Table.Row>
      <Table.Cell>홍길동</Table.Cell>
      <Table.Cell>hong@example.com</Table.Cell>
      <Table.Cell>관리자</Table.Cell>
    </Table.Row>
    <Table.Row>
      <Table.Cell>김철수</Table.Cell>
      <Table.Cell>kim@example.com</Table.Cell>
      <Table.Cell>편집자</Table.Cell>
    </Table.Row>
  </Table.Body>
</Table.Root>
```

## Caption

`Table.Caption`으로 테이블에 설명을 추가할 수 있습니다.

```tsx
<Table.Root>
  <Table.Header>
    <tr>
      <Table.HeaderCell>상품</Table.HeaderCell>
      <Table.HeaderCell>가격</Table.HeaderCell>
    </tr>
  </Table.Header>
  <Table.Body>
    <Table.Row>
      <Table.Cell>노트북</Table.Cell>
      <Table.Cell>1,200,000원</Table.Cell>
    </Table.Row>
  </Table.Body>
  <Table.Caption>2024년 상품 목록</Table.Caption>
</Table.Root>
```

## Hover Row

각 행에는 hover 시 배경색이 변경되는 효과가 기본 적용되어 있습니다.

## Compound Components

| Component | Element | Description |
|-----------|---------|-------------|
| `Table.Root` | `<table>` | 테이블 루트 |
| `Table.Header` | `<thead>` | 헤더 영역 |
| `Table.HeaderCell` | `<th>` | 헤더 셀 |
| `Table.Body` | `<tbody>` | 본문 영역 |
| `Table.Row` | `<tr>` | 행 |
| `Table.Cell` | `<td>` | 셀 |
| `Table.Caption` | `<caption>` | 테이블 설명 |

각 서브 컴포넌트는 대응하는 HTML 요소의 기본 속성을 모두 지원합니다.
