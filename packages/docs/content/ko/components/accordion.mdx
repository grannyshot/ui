# Accordion

접을 수 있는 콘텐츠 섹션 목록을 제공하는 컴포넌트입니다. [Ark UI](https://ark-ui.com/react/docs/components/accordion) 기반으로 구축되었습니다.

## Import

```tsx
import { Accordion } from '@grannyshot/ui'
```

## Usage

```tsx
<Accordion.Root collapsible>
  <Accordion.Item value="item-1">
    <Accordion.ItemTrigger>첫 번째 항목</Accordion.ItemTrigger>
    <Accordion.ItemContent>
      첫 번째 항목의 내용입니다.
    </Accordion.ItemContent>
  </Accordion.Item>
  <Accordion.Item value="item-2">
    <Accordion.ItemTrigger>두 번째 항목</Accordion.ItemTrigger>
    <Accordion.ItemContent>
      두 번째 항목의 내용입니다.
    </Accordion.ItemContent>
  </Accordion.Item>
</Accordion.Root>
```

## Collapsible

`collapsible` prop을 설정하면 열린 항목을 다시 클릭해서 닫을 수 있습니다.

```tsx
<Accordion.Root collapsible>
  <Accordion.Item value="item-1">
    <Accordion.ItemTrigger>닫을 수 있는 항목</Accordion.ItemTrigger>
    <Accordion.ItemContent>다시 클릭하면 닫힙니다.</Accordion.ItemContent>
  </Accordion.Item>
</Accordion.Root>
```

## Multiple

`multiple` prop을 설정하면 여러 항목을 동시에 열 수 있습니다.

```tsx
<Accordion.Root multiple>
  <Accordion.Item value="item-1">
    <Accordion.ItemTrigger>항목 1</Accordion.ItemTrigger>
    <Accordion.ItemContent>내용 1</Accordion.ItemContent>
  </Accordion.Item>
  <Accordion.Item value="item-2">
    <Accordion.ItemTrigger>항목 2</Accordion.ItemTrigger>
    <Accordion.ItemContent>내용 2</Accordion.ItemContent>
  </Accordion.Item>
  <Accordion.Item value="item-3">
    <Accordion.ItemTrigger>항목 3</Accordion.ItemTrigger>
    <Accordion.ItemContent>내용 3</Accordion.ItemContent>
  </Accordion.Item>
</Accordion.Root>
```

## Default Value

`defaultValue`로 초기에 열려있는 항목을 지정합니다.

```tsx
<Accordion.Root defaultValue={['item-2']} collapsible>
  <Accordion.Item value="item-1">
    <Accordion.ItemTrigger>항목 1</Accordion.ItemTrigger>
    <Accordion.ItemContent>내용 1</Accordion.ItemContent>
  </Accordion.Item>
  <Accordion.Item value="item-2">
    <Accordion.ItemTrigger>항목 2 (기본 열림)</Accordion.ItemTrigger>
    <Accordion.ItemContent>이 항목은 기본으로 열려있습니다.</Accordion.ItemContent>
  </Accordion.Item>
</Accordion.Root>
```

## Compound Components

| Component | Description |
|-----------|-------------|
| `Accordion.Root` | 아코디언 루트. `collapsible`, `multiple`, `defaultValue` 등 설정 |
| `Accordion.Item` | 개별 아코디언 항목. `value` 필수 |
| `Accordion.ItemTrigger` | 항목 열기/닫기 버튼 (화살표 아이콘 자동 포함) |
| `Accordion.ItemContent` | 항목 콘텐츠 영역 |

## Props

### Accordion.Root

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `collapsible` | `boolean` | `false` | 열린 항목을 다시 닫을 수 있는지 여부 |
| `multiple` | `boolean` | `false` | 여러 항목 동시 열기 허용 |
| `defaultValue` | `string[]` | — | 초기에 열려있는 항목의 value 배열 |
| `value` | `string[]` | — | 제어 모드에서 열린 항목 |
| `onValueChange` | `(details: { value: string[] }) => void` | — | 항목 변경 시 콜백 |

Ark UI Accordion의 모든 props를 지원합니다.
