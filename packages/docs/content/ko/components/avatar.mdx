# 아바타 (Avatar)

사용자 또는 엔티티를 시각적으로 나타내는 컴포넌트입니다. 이미지, 이니셜 폴백, 다양한 크기를 지원합니다.

## Import

```tsx
import { Avatar } from '@grannyshot/ui'
```

## 사용법

```tsx
<Avatar src="/avatar.jpg" alt="홍길동" />
```

## 이니셜 폴백

`src`가 없거나 이미지 로드에 실패하면, `name` prop에서 이니셜을 자동 생성합니다.

```tsx
<Avatar name="홍길동" />
```

## 크기 (Sizes)

```tsx
<Avatar size="sm" name="SM" />  {/* 32px */}
<Avatar size="md" name="MD" />  {/* 40px */}
<Avatar size="lg" name="LG" />  {/* 48px */}
<Avatar size="xl" name="XL" />  {/* 64px */}
```

## 이미지 사용

```tsx
<Avatar src="/avatar.jpg" alt="김철수" size="lg" />
```

## Props

| Prop | Type | Default | 설명 |
|------|------|---------|------|
| `src` | `string` | — | 이미지 URL |
| `alt` | `string` | — | 이미지 대체 텍스트 |
| `name` | `string` | — | 이니셜 폴백용 사용자 이름 |
| `size` | `'sm' \| 'md' \| 'lg' \| 'xl'` | `'md'` | 아바타 크기 (32/40/48/64px) |
| `className` | `string` | — | 추가 CSS 클래스 |

네이티브 `<div>` HTML 속성을 모두 지원합니다.
