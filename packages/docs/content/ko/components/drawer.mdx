# Drawer

화면 가장자리에서 슬라이드하여 나타나는 패널 컴포넌트입니다. Ark UI Dialog 기반으로 구현되어 있습니다.

## Import

```tsx
import { Drawer, DrawerProvider, drawer } from '@grannyshot/ui'
```

## 설정 (명령형 API)

명령형 API를 사용하려면 앱 루트에 `DrawerProvider`를 추가하세요:

```tsx
<ThemeProvider>
  <App />
  <DrawerProvider />
</ThemeProvider>
```

## 명령형 API

### Open

프로그래밍 방식으로 Drawer를 열 수 있습니다:

```tsx
drawer.open({
  title: '설정',
  description: '환경설정을 관리합니다',
  placement: 'right',
  content: (
    <div>
      <p>설정 콘텐츠</p>
    </div>
  ),
})
```

### Close

현재 열려 있는 Drawer를 닫습니다:

```tsx
drawer.close()
```

## 선언적 API

Compound 컴포넌트를 사용하여 세밀하게 제어할 수 있습니다:

```tsx
<Drawer.Root>
  <Drawer.Trigger asChild>
    <Button>Drawer 열기</Button>
  </Drawer.Trigger>
  <Drawer.Content placement="right">
    <Drawer.Header>
      <Drawer.Title>내비게이션</Drawer.Title>
      <Drawer.Description>페이지 탐색</Drawer.Description>
      <Drawer.CloseTrigger />
    </Drawer.Header>
    <Drawer.Body>
      {/* 콘텐츠 */}
    </Drawer.Body>
    <Drawer.Footer>
      <Button>저장</Button>
    </Drawer.Footer>
  </Drawer.Content>
</Drawer.Root>
```

## 위치 (Placement)

Drawer는 왼쪽 또는 오른쪽에서 슬라이드할 수 있습니다:

```tsx
{/* 왼쪽 */}
<Drawer.Content placement="left">...</Drawer.Content>

{/* 오른쪽 (기본값) */}
<Drawer.Content placement="right">...</Drawer.Content>
```

## Props (Drawer.Content)

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `placement` | `'left' \| 'right'` | `'right'` | Drawer가 나타나는 방향 |
| `className` | `string` | — | 추가 CSS 클래스 |
