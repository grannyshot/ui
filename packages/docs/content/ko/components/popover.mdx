# Popover

트리거 요소 옆에 나타나는 플로팅 패널입니다. 추가 콘텐츠나 액션을 표시할 때 유용합니다.

## Import

```tsx
import { Popover } from '@grannyshot/ui'
```

## 기본 사용법

```tsx
<Popover.Root>
  <Popover.Trigger asChild>
    <Button>Popover 열기</Button>
  </Popover.Trigger>
  <Popover.Content>
    <Popover.Arrow />
    <Popover.Title>상세 정보</Popover.Title>
    <Popover.Description>
      추가 정보가 여기에 표시됩니다.
    </Popover.Description>
    <Popover.CloseTrigger />
  </Popover.Content>
</Popover.Root>
```

## 폼과 함께 사용

인라인 수정이나 작은 폼에 적합합니다:

```tsx
<Popover.Root>
  <Popover.Trigger asChild>
    <Button variant="outline">이름 수정</Button>
  </Popover.Trigger>
  <Popover.Content>
    <Popover.Arrow />
    <Popover.Title>이름 수정</Popover.Title>
    <Input placeholder="이름을 입력하세요" />
    <Button size="sm">저장</Button>
    <Popover.CloseTrigger />
  </Popover.Content>
</Popover.Root>
```

## 화살표 없이 사용

`Popover.Arrow`를 생략하면 플랫한 형태로 표시됩니다:

```tsx
<Popover.Root>
  <Popover.Trigger asChild>
    <Button>정보</Button>
  </Popover.Trigger>
  <Popover.Content>
    <Popover.Title>화살표 없음</Popover.Title>
    <Popover.Description>이 Popover에는 화살표가 없습니다.</Popover.Description>
    <Popover.CloseTrigger />
  </Popover.Content>
</Popover.Root>
```

## Compound 컴포넌트

| 컴포넌트 | 설명 |
|-----------|------|
| `Popover.Root` | 상태 컨테이너 |
| `Popover.Trigger` | Popover를 토글하는 요소 |
| `Popover.Content` | 플로팅 콘텐츠 패널 |
| `Popover.Arrow` | 트리거를 가리키는 화살표 |
| `Popover.Title` | 제목 텍스트 |
| `Popover.Description` | 설명 텍스트 |
| `Popover.CloseTrigger` | 닫기 버튼 |
