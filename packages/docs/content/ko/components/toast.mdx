# 토스트 (Toast)

Sonner 스타일 스택 애니메이션을 지원하는 알림 토스트. 명령형 API로 간편하게 사용할 수 있습니다.

## Import

```tsx
import { ToastProvider, toast } from '@grannyshot/ui'
```

## 셋업

앱 루트에 `ToastProvider`를 추가합니다:

```tsx
<ThemeProvider>
  <App />
  <ToastProvider placement="bottom-end" />
</ThemeProvider>
```

## 명령형 API

### 간편 메서드

```tsx
toast.success('저장되었습니다')
toast.error('문제가 발생했습니다')
toast.warning('주의가 필요합니다')
toast.info('참고 사항입니다')
```

### 설명 포함

```tsx
toast.success({
  title: '파일 업로드 완료',
  description: '파일이 성공적으로 업로드되었습니다.',
})
```

### Promise

비동기 작업의 상태를 자동으로 표시합니다:

```tsx
await toast.promise(uploadFile(), {
  loading: '업로드 중...',
  success: '업로드 완료!',
  error: '업로드 실패',
})
```

### 커스텀 렌더

완전히 자유로운 형태의 토스트를 렌더링합니다:

```tsx
toast.custom((dismiss) => (
  <div style={{ display: 'flex', alignItems: 'center', gap: 12 }}>
    <Avatar size="sm" name="Bot" />
    <div>
      <p>커스텀 콘텐츠</p>
      <button onClick={dismiss}>닫기</button>
    </div>
  </div>
))
```

### 닫기

```tsx
const id = toast.success('안녕하세요')
toast.dismiss(id)   // 특정 토스트 닫기
toast.dismiss()     // 모든 토스트 닫기
```

## 배치 (Placement)

```tsx
<ToastProvider placement="top" />        // 상단 중앙
<ToastProvider placement="top-start" />  // 상단 좌측
<ToastProvider placement="top-end" />    // 상단 우측
<ToastProvider placement="bottom" />     // 하단 중앙
<ToastProvider placement="bottom-start" /> // 하단 좌측
<ToastProvider placement="bottom-end" /> // 하단 우측 (기본값)
```

## Props (ToastProvider)

| Prop | Type | Default | 설명 |
|------|------|---------|------|
| `placement` | `'top-start' \| 'top' \| 'top-end' \| 'bottom-start' \| 'bottom' \| 'bottom-end'` | `'bottom-end'` | 토스트 표시 위치 |

## 변형 (Variants)

토스트 유형에 따라 좌측에 컬러 보더가 표시됩니다:

| 타입 | 색상 |
|------|------|
| `success` | 초록 |
| `error` | 빨강 |
| `warning` | 노랑 |
| `info` | 파랑 |
| 기본 | 테두리 색상 |
