# Select

Ark UI 기반의 드롭다운 셀렉트 컨트롤로, 컬렉션 기반 아이템을 지원합니다.

## Import

```tsx
import { Select, createListCollection } from '@grannyshot/ui'
```

## 사용법

```tsx
const collection = createListCollection({
  items: [
    { label: 'React', value: 'react' },
    { label: 'Vue', value: 'vue' },
    { label: 'Svelte', value: 'svelte' },
  ],
})

<Select collection={collection} placeholder="프레임워크 선택" />
```

## 크기

```tsx
<Select collection={collection} size="sm" placeholder="Small" />
<Select collection={collection} size="md" placeholder="Medium" />
<Select collection={collection} size="lg" placeholder="Large" />
```

## 비활성화

```tsx
<Select collection={collection} placeholder="비활성화" disabled />
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `collection` | `ListCollection` | — | 선택 가능한 아이템 컬렉션 |
| `size` | `'sm' \| 'md' \| 'lg'` | `'md'` | 셀렉트 크기 |
| `placeholder` | `string` | — | 플레이스홀더 텍스트 |
| `disabled` | `boolean` | `false` | 비활성화 상태 |
| `className` | `string` | — | 추가 CSS 클래스 |

### createListCollection

아이템 배열로 컬렉션을 생성합니다.

```tsx
const collection = createListCollection({
  items: [
    { label: '옵션 1', value: 'option-1' },
    { label: '옵션 2', value: 'option-2' },
  ],
})
```
