# DatePicker

Ark UI 기반의 날짜 선택 컨트롤로, 일/월/연 뷰 전환을 지원합니다.

## Import

```tsx
import { DatePicker } from '@grannyshot/ui'
```

## 사용법

```tsx
<DatePicker />
```

## 플레이스홀더

```tsx
<DatePicker placeholder="날짜를 선택하세요" />
```

## 뷰 모드

DatePicker는 세 가지 뷰 모드를 지원하며, 사용자가 자유롭게 전환할 수 있습니다:

- **일 뷰** — 특정 날짜 선택 (기본)
- **월 뷰** — 월 선택
- **연 뷰** — 연도 선택

```tsx
{/* 월 뷰에서 시작 */}
<DatePicker view="month" />

{/* 연 뷰에서 시작 */}
<DatePicker view="year" />
```

## 비활성화

```tsx
<DatePicker disabled />
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `placeholder` | `string` | — | 플레이스홀더 텍스트 |
| `view` | `'day' \| 'month' \| 'year'` | `'day'` | 초기 캘린더 뷰 |
| `disabled` | `boolean` | `false` | 비활성화 상태 |
| `className` | `string` | — | 추가 CSS 클래스 |
