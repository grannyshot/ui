# Tabs

콘텐츠를 탭으로 구분하여 전환할 수 있는 컴포넌트입니다. [Ark UI](https://ark-ui.com/react/docs/components/tabs) 기반으로 구축되었습니다.

## Import

```tsx
import { Tabs } from '@grannyshot/ui'
```

## Usage

```tsx
<Tabs.Root defaultValue="tab1">
  <Tabs.List>
    <Tabs.Trigger value="tab1">개요</Tabs.Trigger>
    <Tabs.Trigger value="tab2">기능</Tabs.Trigger>
    <Tabs.Trigger value="tab3">설정</Tabs.Trigger>
    <Tabs.Indicator />
  </Tabs.List>
  <Tabs.Content value="tab1">개요 내용</Tabs.Content>
  <Tabs.Content value="tab2">기능 내용</Tabs.Content>
  <Tabs.Content value="tab3">설정 내용</Tabs.Content>
</Tabs.Root>
```

## Variants

`variant` prop을 `Tabs.Root`에서 설정하면 하위 컴포넌트에 자동으로 전파됩니다.

### Line (기본값)

하단 테두리 기반의 탭 스타일입니다.

```tsx
<Tabs.Root variant="line" defaultValue="tab1">
  <Tabs.List>
    <Tabs.Trigger value="tab1">탭 1</Tabs.Trigger>
    <Tabs.Trigger value="tab2">탭 2</Tabs.Trigger>
    <Tabs.Indicator />
  </Tabs.List>
  <Tabs.Content value="tab1">Line 스타일 콘텐츠</Tabs.Content>
  <Tabs.Content value="tab2">Line 스타일 콘텐츠</Tabs.Content>
</Tabs.Root>
```

### Pill

배경색 기반의 둥근 탭 스타일입니다.

```tsx
<Tabs.Root variant="pill" defaultValue="tab1">
  <Tabs.List>
    <Tabs.Trigger value="tab1">탭 1</Tabs.Trigger>
    <Tabs.Trigger value="tab2">탭 2</Tabs.Trigger>
  </Tabs.List>
  <Tabs.Content value="tab1">Pill 스타일 콘텐츠</Tabs.Content>
  <Tabs.Content value="tab2">Pill 스타일 콘텐츠</Tabs.Content>
</Tabs.Root>
```

## Disabled

개별 탭을 비활성화할 수 있습니다.

```tsx
<Tabs.Root defaultValue="tab1">
  <Tabs.List>
    <Tabs.Trigger value="tab1">활성</Tabs.Trigger>
    <Tabs.Trigger value="tab2" disabled>비활성</Tabs.Trigger>
    <Tabs.Indicator />
  </Tabs.List>
  <Tabs.Content value="tab1">활성 탭 콘텐츠</Tabs.Content>
</Tabs.Root>
```

## Compound Components

| Component | Description |
|-----------|-------------|
| `Tabs.Root` | 탭 루트. `variant`, `defaultValue` 등 설정 |
| `Tabs.List` | 탭 트리거 목록 컨테이너 |
| `Tabs.Trigger` | 개별 탭 버튼. `value` 필수 |
| `Tabs.Content` | 탭 콘텐츠 패널. `value`로 트리거와 연결 |
| `Tabs.Indicator` | 활성 탭 하단 인디케이터 (line variant에서 사용) |

## Props

### Tabs.Root

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `variant` | `'line' \| 'pill'` | `'line'` | 탭 스타일 (하위 컴포넌트에 자동 전파) |
| `defaultValue` | `string` | — | 초기 활성 탭 값 |
| `value` | `string` | — | 제어 모드에서 활성 탭 값 |
| `onValueChange` | `(details: { value: string }) => void` | — | 탭 변경 시 콜백 |
| `lazyMount` | `boolean` | `true` | 콘텐츠 지연 마운트 |

Ark UI Tabs의 모든 props를 지원합니다.
