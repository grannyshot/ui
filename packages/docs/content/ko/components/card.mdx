# Card

콘텐츠를 시각적으로 그룹화하는 컨테이너 컴포넌트입니다.

## Import

```tsx
import { Card } from '@grannyshot/ui'
```

## Usage

```tsx
<Card>
  <h3>제목</h3>
  <p>카드 안에 들어가는 콘텐츠입니다.</p>
</Card>
```

## Padding

`padding` prop으로 내부 여백을 조절합니다.

```tsx
<Card padding="sm">Small padding</Card>
<Card padding="md">Medium padding (기본값)</Card>
<Card padding="lg">Large padding</Card>
```

## Hoverable

`hoverable` prop을 설정하면 hover 시 테두리와 그림자가 강조됩니다.

```tsx
<Card hoverable>
  클릭 가능한 카드
</Card>
```

## 조합 예시

```tsx
<Card padding="lg" hoverable>
  <h3>프로젝트 이름</h3>
  <p>프로젝트 설명이 여기에 들어갑니다.</p>
</Card>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `padding` | `'sm' \| 'md' \| 'lg'` | `'md'` | 내부 여백 크기 |
| `hoverable` | `boolean` | `false` | hover 시 강조 효과 |
| `className` | `string` | — | 추가 CSS 클래스 |

`React.HTMLAttributes<HTMLDivElement>`의 모든 속성을 지원합니다.
