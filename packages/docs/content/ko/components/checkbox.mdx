# Checkbox

Ark UI 기반의 체크박스 컨트롤로, 다양한 크기와 불확정(indeterminate) 상태를 지원합니다.

## Import

```tsx
import { Checkbox } from '@grannyshot/ui'
```

## 사용법

```tsx
<Checkbox label="약관에 동의합니다" />
```

## 크기

```tsx
<Checkbox size="sm" label="Small" />
<Checkbox size="md" label="Medium" />
<Checkbox size="lg" label="Large" />
```

## 상태

```tsx
<Checkbox label="미선택" />
<Checkbox label="선택됨" defaultChecked />
<Checkbox label="불확정" checked="indeterminate" />
```

## 비활성화

```tsx
<Checkbox label="비활성화" disabled />
<Checkbox label="비활성화 + 선택됨" disabled defaultChecked />
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `label` | `string` | — | 체크박스 레이블 텍스트 |
| `size` | `'sm' \| 'md' \| 'lg'` | `'md'` | 체크박스 크기 |
| `checked` | `boolean \| 'indeterminate'` | — | 제어 모드 체크 상태 |
| `defaultChecked` | `boolean` | `false` | 초기 체크 상태 |
| `disabled` | `boolean` | `false` | 비활성화 상태 |
| `className` | `string` | — | 추가 CSS 클래스 |
