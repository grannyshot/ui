# Progress

진행 상태를 나타내는 프로그레스 바. Ark UI 기반으로 구현되었습니다.

## Import

```tsx
import { Progress } from '@grannyshot/ui'
```

## 기본 사용법

```tsx
<Progress value={40} />
```

## 크기 (Sizes)

```tsx
<Progress size="sm" value={30} />  {/* 4px */}
<Progress size="md" value={50} />  {/* 8px */}
<Progress size="lg" value={70} />  {/* 12px */}
```

## 라벨 표시

```tsx
<Progress value={60} label="업로드 중..." />
```

## 값 표시

```tsx
<Progress value={75} showValue />
```

## 라벨과 값 함께 표시

```tsx
<Progress value={45} label="다운로드 중" showValue />
```

## 최솟값 / 최댓값

```tsx
<Progress value={3} min={0} max={10} />
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `number` | `0` | 현재 진행 값 |
| `min` | `number` | `0` | 최솟값 |
| `max` | `number` | `100` | 최댓값 |
| `size` | `'sm' \| 'md' \| 'lg'` | `'md'` | 바 높이 (sm=4px, md=8px, lg=12px) |
| `label` | `string` | — | 바 위에 표시되는 라벨 텍스트 |
| `showValue` | `boolean` | `false` | 퍼센트 값 표시 여부 |
| `className` | `string` | — | 추가 CSS 클래스 |
