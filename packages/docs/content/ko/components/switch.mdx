# Switch

Ark UI 기반의 토글 스위치 컨트롤로, 다양한 크기를 지원합니다.

## Import

```tsx
import { Switch } from '@grannyshot/ui'
```

## 사용법

```tsx
<Switch label="알림 활성화" />
```

## 크기

```tsx
<Switch size="sm" label="Small" />
<Switch size="md" label="Medium" />
<Switch size="lg" label="Large" />
```

## 기본 활성화

```tsx
<Switch label="기본 활성화" defaultChecked />
```

## 비활성화

```tsx
<Switch label="비활성화" disabled />
<Switch label="비활성화 + 켜짐" disabled defaultChecked />
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `label` | `string` | — | 스위치 레이블 텍스트 |
| `size` | `'sm' \| 'md' \| 'lg'` | `'md'` | 스위치 크기 |
| `checked` | `boolean` | — | 제어 모드 체크 상태 |
| `defaultChecked` | `boolean` | `false` | 초기 체크 상태 |
| `disabled` | `boolean` | `false` | 비활성화 상태 |
| `className` | `string` | — | 추가 CSS 클래스 |
