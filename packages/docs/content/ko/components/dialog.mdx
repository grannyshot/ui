# 다이얼로그 (Dialog)

선언적/명령형 API를 모두 지원하는 모달 다이얼로그 컴포넌트입니다.

## Import

```tsx
import { Dialog, DialogProvider, dialog } from '@grannyshot/ui'
```

## 설정 (명령형 API)

명령형 API를 사용하려면 앱 루트에 `DialogProvider`를 추가하세요:

```tsx
<ThemeProvider>
  <App />
  <DialogProvider />
</ThemeProvider>
```

## 명령형 API

### Confirm

`Promise<boolean>`을 반환합니다. 확인 시 `true`, 취소 시 `false`입니다.

```tsx
const confirmed = await dialog.confirm({
  title: '항목 삭제',
  description: '이 작업은 되돌릴 수 없습니다.',
  confirmText: '삭제',
  cancelText: '취소',
})

if (confirmed) {
  // 삭제 로직
}
```

### Alert

`Promise<void>`를 반환합니다. 확인 버튼을 누르면 resolve됩니다.

```tsx
await dialog.alert({
  title: '완료',
  description: '작업이 성공적으로 완료되었습니다.',
  confirmText: '확인',
})
```

### 커스텀 모달

완전히 자유로운 콘텐츠로 다이얼로그를 열 수 있습니다:

```tsx
dialog.open({
  size: 'lg',
  content: (close) => (
    <div>
      <h2>커스텀 모달</h2>
      <p>원하는 콘텐츠를 자유롭게 배치하세요</p>
      <Button onClick={close}>닫기</Button>
    </div>
  ),
})
```

프로그래밍 방식으로 닫기:

```tsx
dialog.close()
```

## 선언적 API

Compound 컴포넌트를 사용하여 세밀하게 제어할 수 있습니다:

```tsx
<Dialog.Root>
  <Dialog.Trigger asChild>
    <Button>열기</Button>
  </Dialog.Trigger>
  <Dialog.Content size="sm">
    <Dialog.CloseTrigger />
    <Dialog.Title>제목</Dialog.Title>
    <Dialog.Description>설명</Dialog.Description>
    {/* 콘텐츠 */}
  </Dialog.Content>
</Dialog.Root>
```

## Props (Dialog.Content)

| Prop | Type | Default | 설명 |
|------|------|---------|------|
| `size` | `'sm' \| 'md' \| 'lg'` | `'md'` | 다이얼로그 너비 |
| `className` | `string` | — | 추가 CSS 클래스 |
