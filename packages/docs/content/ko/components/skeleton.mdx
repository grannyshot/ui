# 스켈레톤 (Skeleton)

콘텐츠가 로딩 중임을 나타내는 플레이스홀더 컴포넌트입니다. 펄스 애니메이션으로 콘텐츠가 표시될 위치를 보여줍니다.

## Import

```tsx
import { Skeleton } from '@grannyshot/ui'
```

## 사용법

```tsx
<Skeleton width={200} height={20} />
```

## 변형 (Variants)

### 직사각형 (기본값)

```tsx
<Skeleton variant="rectangular" width={200} height={120} />
```

### 텍스트

```tsx
<Skeleton variant="text" width={160} />
```

### 원형

```tsx
<Skeleton variant="circular" width={48} height={48} />
```

## 로딩 상태 구성 예시

```tsx
<div style={{ display: 'flex', alignItems: 'center', gap: 12 }}>
  <Skeleton variant="circular" width={40} height={40} />
  <div>
    <Skeleton variant="text" width={120} />
    <Skeleton variant="text" width={80} />
  </div>
</div>
```

## Props

| Prop | Type | Default | 설명 |
|------|------|---------|------|
| `variant` | `'text' \| 'circular' \| 'rectangular'` | `'rectangular'` | 스켈레톤 형태 |
| `width` | `number \| string` | — | 너비 |
| `height` | `number \| string` | — | 높이 |
| `className` | `string` | — | 추가 CSS 클래스 |

네이티브 `<div>` HTML 속성을 모두 지원합니다.
