# Field

레이블, 힌트 텍스트, 에러 메시지를 제공하고, 하위 폼 컨트롤에 ARIA 속성을 자동 연결하는 폼 필드 래퍼입니다.

## Import

```tsx
import { Field } from '@grannyshot/ui'
```

## 사용법

```tsx
<Field label="사용자명">
  <Input placeholder="사용자명을 입력하세요" />
</Field>
```

## 힌트 텍스트

```tsx
<Field label="비밀번호" hint="8자 이상이어야 합니다">
  <Input type="password" />
</Field>
```

## 필수 필드

```tsx
<Field label="이메일" required>
  <Input type="email" placeholder="you@example.com" />
</Field>
```

## 에러 상태

```tsx
<Field label="사용자명" error="이미 사용 중인 사용자명입니다">
  <Input />
</Field>
```

## 비활성화

```tsx
<Field label="이름" disabled>
  <Input placeholder="비활성화된 필드" />
</Field>
```

## 다양한 컨트롤과 함께 사용

Field는 모든 폼 컨트롤 컴포넌트와 함께 사용할 수 있습니다.

```tsx
<Field label="프레임워크" required>
  <Select collection={collection} placeholder="프레임워크 선택" />
</Field>

<Field label="자기소개" hint="본인에 대해 알려주세요">
  <Textarea placeholder="내용을 입력하세요..." />
</Field>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `label` | `string` | — | 필드 레이블 텍스트 |
| `hint` | `string` | — | 컨트롤 아래 도움말 텍스트 |
| `error` | `string` | — | 에러 메시지 (존재 시 힌트 대체) |
| `required` | `boolean` | `false` | 레이블에 필수 표시 |
| `disabled` | `boolean` | `false` | 필드 및 하위 컨트롤 비활성화 |
| `children` | `ReactNode` | — | 폼 컨트롤 (Input, Select, Textarea 등) |
| `className` | `string` | — | 추가 CSS 클래스 |

Field는 레이블, 힌트, 에러 메시지를 하위 폼 컨트롤에 ARIA 속성(`aria-describedby`, `aria-invalid` 등)으로 자동 연결합니다.
