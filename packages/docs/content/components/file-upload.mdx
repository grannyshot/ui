# FileUpload

A file upload component with drag & drop support. Built on [Ark UI](https://ark-ui.com/react/docs/components/file-upload).

<ComponentPreview name="file-upload" />

## Import

```tsx
import { FileUpload } from '@grannyshot/ui'
```

## Usage

```tsx
<FileUpload label="Attachments" maxFiles={3} />
```

## API Reference

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `label` | `string` | тАФ | Label text |
| `maxFiles` | `number` | тАФ | Maximum number of files |
| `maxFileSize` | `number` | тАФ | Max file size in bytes |
| `accept` | `Record<string, string[]>` | тАФ | Accepted file types |
| `disabled` | `boolean` | `false` | Disabled state |
| `className` | `string` | тАФ | Additional CSS classes |
