# Getting Started

## Installation

```bash
npm install @grannyshot/ui
# or
pnpm add @grannyshot/ui
# or
yarn add @grannyshot/ui
```

## Setup

### 1. Import Styles

Import the CSS file once at the root of your application:

```tsx
import '@grannyshot/ui/styles.css'
```

### 2. Dark Mode (Optional)

3가지 방법 중 택 1:

**Option A: ThemeProvider (가장 간단)**

```tsx
import { ThemeProvider } from '@grannyshot/ui'

export default function App() {
  return (
    <ThemeProvider defaultTheme="system">
      {/* your app */}
    </ThemeProvider>
  )
}
```

**Option B: Blocking Script (Next.js, SSR — flash 없음)**

```tsx
// app/layout.tsx
import { ThemeScript } from '@grannyshot/ui'

export default function RootLayout({ children }) {
  return (
    <html lang="ko">
      <head><ThemeScript /></head>
      <body>{children}</body>
    </html>
  )
}
```

**Option C: 직접 설정**

```html
<!-- data-theme 속성만 세팅하면 됩니다 -->
<html data-theme="dark">
  ...
</html>
```

### 3. Add Providers (Optional)

명령형 API (toast, dialog, drawer)를 사용하려면 Provider를 추가하세요:

```tsx
import { ToastProvider, DialogProvider, DrawerProvider } from '@grannyshot/ui'

export default function App() {
  return (
    <>
      {/* your app */}
      <ToastProvider placement="bottom-end" />
      <DialogProvider />
      <DrawerProvider />
    </>
  )
}
```

### 4. Use Components

```tsx
import { Button, Input, Field } from '@grannyshot/ui'

function LoginForm() {
  return (
    <form>
      <Field label="Email" required>
        <Input placeholder="you@example.com" />
      </Field>
      <Button variant="primary">Sign in</Button>
    </form>
  )
}
```

## Using with Tailwind CSS

grannyshot-ui는 Tailwind CSS로 빌드되어 있어서 **Tailwind 프로젝트에서 충돌 없이** 바로 사용할 수 있습니다.

```tsx
// Tailwind 프로젝트에서도 그냥 사용
import '@grannyshot/ui/styles.css'
```

컴포넌트에 Tailwind 클래스를 추가하면 자연스럽게 오버라이드됩니다:

```tsx
// grannyshot-ui 기본 스타일 유지
<Button variant="primary">Submit</Button>

// Tailwind로 오버라이드 가능
<Button variant="primary" className="rounded-full">Submit</Button>

// Tailwind 레이아웃 안에 grannyshot-ui 컴포넌트
<div className="grid grid-cols-2 gap-4 p-6">
  <Card>...</Card>
  <Card>...</Card>
</div>
```

### CSS 파일 비교

| 파일 | 용도 |
|------|------|
| `styles.css` | 기본 — Tailwind 프로젝트 또는 단독 사용 |
| `styles-no-preflight.css` | 다른 CSS 프레임워크와 함께 사용 (Tailwind reset 제거) |

## Import Paths

| Path | Description |
|------|-------------|
| `@grannyshot/ui` | All components, styles, tokens, and utilities |
| `@grannyshot/ui/react` | React components only |
| `@grannyshot/ui/styles` | Style recipes (tv) only |
| `@grannyshot/ui/tokens` | Design tokens |
| `@grannyshot/ui/context` | ThemeProvider, ThemeScript, and useTheme |
| `@grannyshot/ui/utils` | cn utility |
| `@grannyshot/ui/styles.css` | CSS file |
| `@grannyshot/ui/styles-no-preflight.css` | CSS file (reset 제거) |
